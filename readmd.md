# سيرتي

موقع سيرتي الذاتية.

## مهام

* إصلاح مشكلة حجم عرض المشهد الأيقوني.
* إضافة مركب خبرة.
* إكمال صفحة الخبرة.
* إضافة مركب مشروع.
* إكمال صفحة المشاريع.
* إكمال صفحة الاتصال.
* تشطيبات.
* إضافة النسخة الإنجليزية.
* تحميل الموقع.
* دعم نمطين فاتح وداكن.
* إضافة سكريبت لتحديث الموقع تلقائيًا.
* تحديث منصة الويب لتحرير المستخدم من كتابة شفرة CSS داخل برنامج الأسس.
